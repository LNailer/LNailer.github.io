Languages

I'm just using this page to check language formatting on GitHub.

Thai • ภาษาไทย - Thai script has similarities with Old Khmer.
Vietnamese • Tiếng Việt - Roman script with a lot of diacritics.
Khmer (Cambodian) • ភាសាខ្មែរ - Written in the Khmer script.
Burmese • မြန်မာဘာသာ - Written in Burmese script.
Korean • 한국어 - Written in Hangul - Korean script.
Mandarin • 普通话 - Written in Simplified Chinese in mainland China.
Urdu • اُردُو - Written with an extended Persian alphabet.
Farsi • فارسی - Written in the Persian alphabet.
Dari • دری - A lot like Farsi, same alphabet but used in Tajik and Hazara communities.
Hindi • हिन्दी - Written in the Devanagari script.
Pashto • پښتو - Written with a modified version of the Persian script with extra letters.
